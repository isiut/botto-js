# botto-js

A Discord bot made with JavaScript and discord.js for the purpose of practicing coding

---

## Planned features

-   Magic 8-ball (done)
-   Memes from reddit
-   Unit conversions
-   Weather data
-   Dictionary lookup
-   Rock Paper Scissors
-   Higher or lower number game
-   Random username generator
-   Moderation commands
-   And other small things ...
